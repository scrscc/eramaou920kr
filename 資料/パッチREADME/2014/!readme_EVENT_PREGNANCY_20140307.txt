■仕様
2014/03/07
妊娠時のストレス値の分岐修正。
父親が野良犬でなおかつ結婚相手が野良犬の場合ストレスが減るようにしました。
父親がモンスターで母親が魔族の場合ストレスが減るようにしました。
父親がモンスターで母親の結婚相手がモンスターの場合ストレスが減るようにしました。

2013/02/25
奴隷同士での妊娠期間を短縮。
女性あなたがモンスターとの性交で妊娠するようになりました。

2013/02/24
奴隷同士での妊娠でもモンスターを産むようにしました。
女性あなたが妊娠するようになりました。
逆レイプだと童貞喪失しないを修正。
助手を犯させるで妊娠しなかったのを修正。

2013/01/18
奴隷同士で妊娠した場合、相手の父親の名前を覚えるようにしました。
父親があなたなら必ず近衛兵を産むように修正

2013/01/05
妊娠関連のERBを見直して
マスター、奴隷同士（助手←→奴隷）、野良犬、コロシアムモンスター、触手の膣内射精で妊娠可能にしました。
キャラ管理の都合上、奴隷同士で妊娠させたあと父親の奴隷を処刑（削除）すると出産時の父親の名前が他の奴隷の名前になるので注意してください。
