このパッチは、eramaouの元勇者売却価格に影響するバグを修正するためのパッチです。

;***************************
;	対象バージョン
;***************************

eramaou v0.304

;***************************
;	適用方法
;***************************

同梱のERBファイルをeramaouのERBフォルダに上書きコピーしてください

;***************************
;	修正内容および修正個所
;***************************

●職業：肉便器での処女V調教が可能になっている

	職業：肉便器の「最下層モンスター奉仕」では
	【貞操封印】を施していている場合のみ処女でV調教が実行されてしまう。
	
	処女のままV調教が出来ると売価が想定以上に上昇する。
	
	○BENKI.ERB(73行目)の、元の条件分岐は以下の通り
		IF TALENT:A:0 == 0 || TALENT:A:273 == 1
	
	地の文の分岐条件が
		IF TALENT:A:0 == 1 || TALENT:A:273 == 1
		（処女、もしくは貞操封印アリの場合A調教のみ）
	なので、その反対の
		IF TALENT:A:0 == 0 && TALENT:A:273 == 0
		（非処女でかつ貞操封印なしの場合、V調教に進む。）
	に修正
	

●記憶消去薬が実行できる

	実験室で直接「8」を入力すると記憶消去薬のメニューに入れてしまう。
	実行した場合、キャラ番号最後尾のキャラに記憶消去薬を使用したキャラの初期状態が書き込まれ
	それ以前のキャラのステータスがひとつづつズレる。
	
	記憶消去自体は実行されているので、CVBALv10が可能になってしまう。
	
	○SHOP_LABO.ERB(58行目)以降をコメントアウト
		;ELSEIF RESULT == 8
		;	CALL MODIFY_AMNESIA
